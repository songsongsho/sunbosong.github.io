Disqus Blog，一个最好用的博客评论平台

2020-12-25 · 2 min read
教程

我们常常在浏览一些博客文章时，对文章内容有着自己的看法想及时反馈给作者时，可以在文章底部看到类似这样的评论留言功能。

基本上很多这种留言都是使用 Disqus 去创建的，由于我的博客是使用 Gridea 配合搭建的，本身它就可以配置 Disqus 账号远程连接；用了一段时间后感受到了 Disqus 的确是一个比较友好博客评论平台。

注册及配置 Disqus

前往 Disqus 官网 找到 Sign up 填写注册信息

完成注册后，进入首页点击 GET STARTED 进入下一步

选择 I want to install Disqus on my site

填写自己的网站信息，输入域名即可。

选择订阅计划（可以免费体验 30 天）

由于我的博客是使用 Gridea 配合搭建的，这里可以先不用去选择。

前往 Disqus API Resources 新建一个 Application

将基础信息完善后，点击 Redister my application 确认

基础信息完善后，进入API 信息页面，在「 Details 」中获取你的 API Keys 与 API Secrets 并复制到剪贴板和其他文本上

接着切换到「 Settings 」中 Domains ，设置域名白名单防止其他用户滥用你的 API ，还能防止某些安全隐患。

信息全部完善后，直接打开已安装的 Gridea 应用找到「 远程 」中 评论配置。

输入 Disqus 创建账号时填写的 Shortname ，同时复制 API 信息页面中的 API Keys 与 API Secrets 对应 Gridea 中的 apikey 和 api ，确认后保存即可。

点击应用左下角的「 同步 」，等待同步完成后访问自己的博客文章就可以看到文章最底部的 Disqus 评论功能。

Disqus 评论数据

前往 Disqus 中的 Analytics 数据中心可以查看用户在访问博客文章时的留言评论信息，可帮助博主更好地了解受众与互动内容。
